# Hardware specification

## Board​

- Arduino zero board to drive the motor
- L298N driver
- Stepper motor ref. MT-1704HS168A

## Luos network on arduino board​

- PTPA: PIN19
- PTPB: PIN7
- RX_EN: 2
- TX_EN: 3
- TX: PIN_SERIAL1_TX
- RX: PIN_SERIAL1_RX
